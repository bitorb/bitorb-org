* Mac OSX 系统下的安装及配置
  Mac OSX 系统默认已经安装好了 apache ，可以直接用下面的命令启动。
#+BEGIN_EXAMPLE
$ which apachectl
/usr/sbin/apachectl
$ sudo apachectl start
#+END_EXAMPLE
  默认的配置中的 DocumentRoot 指向的是 /Library/WebServer/Documents，可以修改为我的任意一个目录。
  同时修改相应的 Directory。
#+BEGIN_EXAMPLE
$ sudo vi /etc/apache2/httpd.conf
$ sudo apachectl start
#+END_EXAMPLE
