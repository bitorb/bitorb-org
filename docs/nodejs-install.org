* Mac OSX 系统下的安装
  先确保已 [[./system-homebrew.org][安装 Homebrew]]，然后再安装 node.js。
#+BEGIN_EXAMPLE
$ brew install node
#+END_EXAMPLE

* 使用 npm 进行其它软件包安装
#+BEGIN_EXAMPLE
$ npm install -g bower
#+END_EXAMPLE
